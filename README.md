# Gestão de Atendimento de Vendas

Este projeto é uma aplicação web simples para gerenciar o atendimento de vendedores em um ambiente comercial. Ele permite adicionar vendedores, movê-los para a lista de atendimento, registrar os resultados dos atendimentos, e calcular a taxa de conversão de vendas. O projeto também inclui funcionalidades como edição dos nomes dos vendedores e exclusão de vendedores por meio de uma interface intuitiva.

## Funcionalidades

- **Adicionar Vendedor**: Permite adicionar novos vendedores à lista de espera.
- **Mover para Atendimento**: Movimenta os vendedores da lista de espera para a lista de atendimento ativo.
- **Registrar Resultado do Atendimento**: Registra se o atendimento foi bem-sucedido, não convertido ou resultou em uma troca.
- **Calcular Taxa de Conversão**: Calcula e exibe a taxa de conversão de vendas com base nos atendimentos registrados.
- **Editar Nome do Vendedor**: Possibilidade de editar o nome dos vendedores diretamente na lista.
- **Excluir Vendedor**: Exclui vendedores da lista de espera ao arrastá-los para a área de exclusão.

## Estrutura do Projeto

- **HTML**: A estrutura principal da página.
- **CSS**: Estilos para tornar a aplicação visualmente atraente e responsiva.
- **JavaScript**: Funções para manipulação de dados, interações do usuário, e armazenamento local.

## Como Usar

1. Clone ou baixe este repositório.
2. Certifique-se de que os caminhos para o arquivo CSS (`css/styles.css`) e o arquivo JavaScript (`js/script.js`) estejam corretos.
3. Abra o arquivo `index.html` em um navegador da web.
4. Adicione vendedores e mova-os entre as listas conforme necessário.
5. Use o botão "Taxa de Conversão" para calcular a eficiência dos vendedores.

## Requisitos

- **Navegador**: O projeto é compatível com a maioria dos navegadores modernos.
- **JavaScript Ativado**: Certifique-se de que o JavaScript esteja ativado no navegador para que todas as funcionalidades funcionem corretamente.

## Personalização

- **Funcionalidade**: O JavaScript pode ser ajustado ou expandido para incluir mais funcionalidades ou adaptar o comportamento existente.
